# kcxy

Rust restful webapp with rocket, diesel, postgresql, and responsive design.

### Run kcxy

```bash
$ git clone https://github.com/mcux/kcxy.git
$ cd kcxy
$ cargo install diesel_cli --no-default-features --features postgres
$ diesel setup
$ cargo run
```

### Run wapp

```bash
$ cd wapp
$ npm install
$ npm run dev
```
